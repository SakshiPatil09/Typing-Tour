ABOUT US

(1). Introduction

We aim to create a typing tour webapp which has user authentication features.
it has courses with different difficulty levels which will help the user to learn and test
their typing skills.
the test has a real time sync system, i.e. as soon as the user starts typing something,
the system will be analysing it in real time
and once the test ends, the user will receive a score based on
their performance along with the complete statistical data which can be exported if the user wishes so.

(2). Structure

The architecture or the framework of the webapp can be explained in terms of frontend and backend.
the interaction between user, frontend and the backend is what the architecture of this webapp is based on.
the user interacts with the frontend which displays the several pages like homepage, login page, about us page, etc.
the frontend is designed using html,css and javascript languages. On the other hand,
the backend handles tasks like calculation, database management and rendering webpages.
the framework of the backend is based on Django.

(3). Our Motive

To teach people how to type in a professional manner with help of techniques like 10 finger
typing enabling them to achive productivity.
this webapp will act as a education tool for all people who work with
electronic devices and require typing as a skillset.The uses of this webapp are:
a typing test for interviews, home schooling, competitive typing test,etc.

(4). Other Updates

We like to improvise the tests part of our webapp even more to make the results more accurate and helpful.
We also plan to commercialize this app by providing promo codes and offers as a
reward for achieving a certain score in the typing tests
